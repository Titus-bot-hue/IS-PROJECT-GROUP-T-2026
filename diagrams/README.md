# Diagrams for Smart School Information System (SSIS)

## 1. Architecture Diagram
Illustrates 3-tier architecture:
- Presentation Layer (Web UI for Admin, Teacher, Student, Parent)
- Application Layer (Business logic, API server)
- Data Layer (Database server, cloud backup)
- Network zones (Internal LAN, DMZ, Internet-facing)

## 2. Data Flow Diagram (DFD)
Shows how data flows between:
- Users → Web Application → Database → Reports

## 3. Entity-Relationship Diagram (ERD)
- Student, Parent, Teacher, Fees, Attendance, Results tables
- Relationships between tables
